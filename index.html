There are many aspects of HTML that are needed to present a web page. There are only a few aspects that can style HTML code and these are reffered to as inline styles. CSS is a better platform to apply these styles. Here is a table to show a few of them. This table is also created using only HTML. Relevent CSS has only been applied to the 'CSS Effect and Benefit' column.

HTML elements

HTML Element Effect Identifiable Issue CSS Alternative CSS Effect and Benefit
<b> Bold text. This has to applied to each individual element. element{
font-weight:bold;
}		 This applies the bold font to all named selectors in bulk. Faster and easier!
<i> or <em> Italics This has to be applied to each individual element. element{
font-style:italic;
}		 This applies the italic font to all named selectors in bulk. Faster and easier!
<element style= 'color: red'> Colors text This is an inline style that is added to each element. element{
color:red;
}		 This applies the colored font to all named selectors in bulk. Faster and easier!
Background color Colours the background of the element This is an inline style that is added to each elelment. element{
background-color:aqua;
}		 This applies the color to all named elements in bulk. Faster and easier!

CSS

Css is far better suited to adjusting the visual presentation of a web page. The attributes can be added to a lot of different elements with only one command as apposed to each individual element in it's line of code. CSS can also use different data packs available to link to or download to increase the visual variety available.

Function CSS Code Outcome
Change the style of font element{
font-family: Arial;
}		 The font is changed
Fonts can be downloaded and linked to in the CSS file @font-face {
font-family: Abel;
src: url(abel/abel-regular.ttf)
}		 The font is declared as above, this changes it from the default available to a web font.
